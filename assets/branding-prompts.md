# Branding Prompts (Banners + Icon)

Use these prompts with your preferred image model to generate professional project branding assets.

## Banner A (README Hero)

- **Filename:** `assets/minecombat-banner-hero.png`
- **Recommended size:** `1792x1024`
- **Prompt:**

Create a modern, professional wide banner for an open-source research toolkit. Minecraft-inspired combat benchmark arena at sunrise, realistic block-based environment, subtle transparent HUD overlays showing benchmark metrics (latency line chart, success rate, episode count), clean composition with negative space for typography, no childish/cartoon style. Color grade in slate navy and steel blue with cyan UI accents and soft amber rim light. Render exact title text: 'MineCombat-Evaluation'. Subtitle text: 'Open Benchmarking Toolkit for Minecraft Combat Agents'. Sharp, production-ready, high contrast, minimal clutter, editorial technology branding. Wide 16:9.

## Banner B (Technical Variant)

- **Filename:** `assets/minecombat-banner-tech.png`
- **Recommended size:** `2048x1024`
- **Prompt:**

Design a professional open-source project banner in a modern technical style, combining a clean benchmark dashboard and an isometric Minecraft-like combat test environment. Left side: UI cards with suite names, reproducibility indicators, performance bars, and run metadata. Right side: controlled combat map lanes with block terrain and test entities, visually precise and research-oriented. Keep typography crisp and central. Use charcoal and graphite base with electric blue and mint accents. Render exact title text: 'MineCombat-Evaluation'. Optional subtitle: 'Reproducible Combat Benchmarks for Open Research'. Avoid playful or childish aesthetics; keep it enterprise/research polished. Wide 2:1 layout.

## Icon

- **Filename:** `assets/minecombat-icon.png`
- **Recommended size:** `1024x1024`

Create a modern app icon for 'MineCombat-Evaluation': a geometric shield fused with a pixel-block cube and a subtle line-chart/checkmark motif to represent combat benchmarking and reproducibility. Flat-to-semi-flat vector style, professional open-source branding, no mascots, no childish look. Palette: dark graphite background, cobalt blue primary, cyan accent, white highlights.
